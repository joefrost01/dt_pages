Dynamic Tuning    

?

To create a new set of parameter/threshold edits, click on the "create change set" button and select the asset class, scenario and product that you want to tune in order create a new "change set" of edits.

You can also add comments to the change set and, when you have executed successful scoring tests, attach the scoring files as evidence.  Once you are happy with a change set you can submit it for approval.  The approvals button will indicate how many approvals are pending (by default you will only see this for your preferred asset class), clicking it will take you to the approvals page.

Note that a change set can only apply to a single asset-class/scenario/product combination.  You can have multiple open change sets at the same time, this can be viewed by selecting "Change sets" from the dropdown menu.  By default you will see only your sets but you can switch to seeing all of them by toggling the user/all buttons.
